# VSCode Amiga E Syntax Highlighting Extension (incl. E-VO compiler)

## Features

- syntax highlighting of all known Amiga E keywords (incl. E-VO)
- adjusted vscode's default dark color theme
- code snippets
- hover information for builtin functions

## Requirements

- none (until now)

## Extension Settings

- none (until now)

## Known Issues

- Syntax Highlighting is not very specific currently. That allows to handle more dynamic cases but can lead e.g. to colored keywords that are not make sense in certain code areas.
